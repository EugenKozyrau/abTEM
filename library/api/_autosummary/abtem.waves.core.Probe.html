Probe#

class abtem.waves.core.Probe(semiangle_cutoff=None, extent=None, gpts=None, sampling=None, energy=None, taper=2.0, tilt=(0.0, 0.0), device=None, aperture=None, aberrations=None, extra_transforms=None, **kwargs)[source]#

Bases: _WavesFactory

The probe represents a stack of electron probe wave functions for simulating scanning transmission electron microscopy.

Parameters

  • semiangle_cutoff (float) – The cutoff semiangle of the aperture [mrad].

  • extent (two float, optional) – Lateral extent of wave functions [Å].

  • gpts (two ints, optional) – Number of grid points describing the wave functions.

  • sampling (two float, optional) – Lateral sampling of wave functions [1 / Å].

  • energy (float, optional) – Electron energy [eV].

  • taper (float, optional) – Taper the edge of the default aperture [mrad]. Default is 2.0.

  • tilt (two float, two 1d Distribution, 2d Distribution, optional) – Small angle beam tilt [mrad].

  • device (str, optional) – The probe wave functions will be build and stored on this device.

  • aperture (BaseAperture) – A custom aperture, the provided aperture should inherit from BaseAperture.

  • aberrations (dict or Aberrations) – The phase aberrations as a dictionary where each

  • extra_transforms (list of WaveTransform) – A list of wave function transforms which will be applied after the aperture, but before any additional transforms, .e.g. scans.

  • kwargs – Provide the aberrations as keyword arguments. The arguments will be forwarded to the Aberrations wave function transform.

Parameters

  • scan (scan object or array of xy-positions) – Positions of the probe wave functions

  • max_batch (int, optional) – The number of wave functions in each chunk of the Dask array. If None, the number of chunks are automatically estimated based on “dask.chunk-size” in the user configuration.
