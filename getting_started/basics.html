Execute computation#

abTEM is lazily evaluated. The result from a computation isn’t computed until you ask for it. Instead, a Dask task graph for the computation is produced.

The array is a Dask Array, this means that it is “lazily” evaluated. When we create a lazy array, we only set up the tasks to compute the array, to execute those tasks we also have to call compute. After calling compute the Dask array becomes a NumPy array.
